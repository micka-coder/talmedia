### Partie 2
ยง La mishna enseigne : Le pretre <b>assemble le grand arrangement</b> sur le cote oriental de l'autel, et son ouverture se trouvait sur le cote oriental de l'autel. La Gemara demande : <b>Quelle est la raison</b> pour laquelle une ouverture a ete formee dans l'arrangement ? Cette question fait l'objet d'une dispute entre <b>Rav Huna et Rav Hisda : L'un dit</b> que l'ouverture a ete formee <b>pour que le vent y souffle</b> et attise les flammes de l'arrangement. <b>Et l'un a dit</b> qu'elle etait formee <b>pour que</b> les pretres <b>puissent</b> allumer de la le petit bois</b> qui etait place entre les buches.

### Partie 3
La Guemara <b>souleve une objection</b> de la mishna : <b>Il y avait un espace entre les buches, car</b> les pretres <b>allumaient le petit bois de la.</b> De toute evidence, le petit bois n'etait pas allume par l'ouverture. La Guemara explique : Le Sage qui soutient que le petit bois etait allume a partir de l'ouverture a l'est de l'arrangement pourrait <b>vous dire</b> qu'en plus d'allumer le petit bois a partir de l'ouverture, les pretres <b>faconnaient de nombreux espaces</b> dans l'arrangement, a travers lesquels ils allumaient le petit bois afin d'augmenter le feu.
