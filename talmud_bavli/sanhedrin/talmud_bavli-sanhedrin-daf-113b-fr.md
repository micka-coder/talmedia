### Partie 2
§ La mishna enseigne en ce qui concerne le verset : <b>"Et il ne restera rien de ce qui a ete consacre a ta main"</b> (Deuteronome 13:18), <b>aussi longtemps que les mechants</b> existent <b>dans le monde,</b> il y a <b>du malheur dans le monde.</b> La Gemara demande : <b>Qui sont</b> ces <b>mechants</b> mentionnes dans la mishna ? <b>Rav Yosef repond : </b> Ce sont des <b>mechants.</b>

### Partie 3
<b>Les Sages ont enseigne</b> dans une <i>baraita</i> : Quand <b>un mechant</b> personne <b>arrive dans le monde, la colere arrive dans le monde, comme il est dit : "Quand le mechant vient dans le monde, le mepris vient aussi, et avec l'ignominie, l'opprobre"</b> (Proverbes 18:3). Quand <b>un mechant</b> est elimine du monde, le bien vient dans le monde, comme il est dit : "Et quand les mechants perissent, il y a jubilation"</b> (Proverbes 11:10). Quand <b>un juste</b> <b>sort du monde, le mal entre dans le monde, comme il est dit : "Le juste perit et personne ne le prend a cœur ; et les hommes misericordieux sont pris, personne ne comprend qu'a cause du mal le juste est pris"</b> (Isaie 57:1) Quand <b>un juste</b> vient dans le monde, le bien vient dans le monde</b> avec lui, <b>comme il est dit</b> a propos de Noe : <b>Voici celui qui nous consolera de notre travail et du labeur de nos mains</b> (Genese 5:29).
