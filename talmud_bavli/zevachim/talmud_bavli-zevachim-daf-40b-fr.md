En ce qui concerne le terme <b>"dans le sang"</b> (Lévitique 4:6), il enseigne <b>que le sang</b> dans le récipient de service <b>doit être d'une</b> <b>mesure suffisante pour l'immersion dès le départ.</b> Le prêtre doit initialement recueillir dans le récipient suffisamment de sang pour toutes les aspersions, plutôt que d'ajouter du sang dans le récipient pour chaque aspersion. Enfin, l'expression <b>"et</b> le prêtre <b>s'immergera</b> son doigt dans le sang" indique qu'il doit y avoir suffisamment de sang dans le récipient pour que le prêtre puisse y plonger son doigt <b>et ne pas</b> avoir à <b>essuyer</b> les côtés de l'ustensile pour recueillir le sang pour l'aspersion.

### Partie 2
La Gemara explique : <b>Et</b> il <b>était nécessaire</b> que le verset énonce ces deux derniers termes. Il fallait <b>écrire "dans le sang", comme si le Miséricordieux avait écrit</b> seulement <b>"et</b> le prêtre <b>s'immergera", je dirais</b> que les aspersions sont valables <b>même s'il n'y avait pas</b> une <b>mesure</b> suffisante de sang <b>pour une immersion dès le départ,</b> mais seulement assez pour une seule aspersion, à condition que le prêtre ajoute ensuite plus de sang au récipient pour chaque aspersion. Par conséquent, <b>Le Miséricordieux écrit "dans le sang"</b> pour enseigner que, dès le départ, il doit y avoir suffisamment de sang dans le récipient pour toutes les aspersions.

### Partie 3
<b>Et</b> inversement, <b>si le Miséricordieux avait écrit</b> seulement <b>"dans le sang", je dirais</b> que <b>même</b> s'il y avait assez de sang dans le récipient au départ, il n'est pas nécessaire qu'il y ait assez de sang pour l'immersion pour les dernières aspersions, car le prêtre peut <b>essuyer</b> les côtés de l'ustensile pour recueillir le sang pour l'aspersion. Par conséquent, <b>Le Miséricordieux écrit "et</b> le prêtre <b>s'immergera."</b>

### Partie 4
La Gemara aborde une autre phrase apparemment superflue dans le même chapitre : "Et le prêtre mettra une partie du sang sur les coins de <b>l'autel des parfums odorants</b> devant l'Eternel" (Lévitique 4:7). <b>Pourquoi ai-je</b> besoin du verset pour mentionner l'encens doux ? Il aurait suffi d'identifier l'autel comme étant "devant le Seigneur" et on aurait compris que la référence est à l'autel intérieur. Cela sert plutôt à enseigner <b>que si l'autel n'avait pas été inauguré avec de l'encens doux,</b> le prêtre <b>n'y aurait pas aspergé</b> du sang.

### Partie 5
La Gemara commente : <b>Il est enseigné</b> dans une <i>baraita</i> <b>conformément</b> à l'avis <b>du Rav Pappa</b> que la phrase superflue dans le passage discutant du taureau pour un péché communautaire involontaire fait allusion au taureau de Yom Kippour afin d'enseigner les trois <i>halakhot</i> de <i>et</i>, dans le sang, et de l'immersion, à partir du cas du taureau pour un péché involontaire du prêtre oint, auquel il est également fait allusion dans ce verset. Le <i>baraita</i> indique : <b>"Et il fera</b> avec le taureau, <b>comme il a fait</b> avec le taureau pour le sacrifice pour le péché" (Lévitique 4:20). <b>Quel est le sens</b> quand <b>le verset dit "avec le taureau" ?</b> Cela sert <b>à inclure le taureau de Yom Kippour pour tout ce qui est énoncé dans</b> cette <b>matière,</b> c'est-à-dire dans le passage concernant le taureau pour un péché involontaire du prêtre oint, spécifiquement les <i>halakhot</i> de <i>et</i>, dans le sang, et l'immersion. C'est <b>la déclaration de Rabbi</b> Yehuda HaNasi.

### Partie 6
<b>Rabbi Yishmael a dit :</b> Cette inclusion est inutile, car ces <i>halakhot</i> peuvent être dérivées via <b>une <i>a fortiori</i></b> inférence : <b>Comme dans un cas où</b> une <b>offrande n'est pas assimilée à</b> une autre <b>offrande</b> d'un type d'animal différent, la Torah a <b>égalisé</b> les <b>actions</b> d'aspersion du sang dans une offrande <b>avec</b> les <b>actions</b> du sang dans l'autre offrande, comme cela sera expliqué, dans <b>un cas dans lequel</b> la Torah <b>égalise</b> une <b>offrande avec</b> une autre <b>offrande,</b> c'est-à-dire. e., le taureau pour un péché involontaire du prêtre oint et le taureau de Yom Kippour, <b>n'est-il pas logique que</b> la Torah <b>assortisse</b> les <b>actions</b> d'aspersion du sang dans une offrande <b>avec</b> les <b>actions</b> du sang dans l'autre offrande ? Par conséquent, la dérivation par le biais d'une inclusion spéciale n'est pas nécessaire.

### Partie 7
<b>Au contraire, quel</b> est le sens lorsque <b>le verset indique :</b> "Et il fera avec le taureau, comme il a fait avec le taureau pour le sacrifice pour le péché" ? En ce qui concerne la première occurrence de <b>"avec le taureau", il s'agit du taureau pour un péché communautaire involontaire.</b> Et en ce qui concerne la deuxième occurrence de <b>"avec le taureau", il s'agit du taureau pour</b> un péché involontaire du <b>prêtre oint.</b> Et le verset sert à enseigner que tout comme dans le premier cas, si le prêtre a omis une des aspersions, il n'a rien fait, il en est de même dans le deuxième cas.

### Partie 8
La Guemara clarifie la <i>baraita</i>. <b>Le Maître a dit</b> ci-dessus : <b>Comme dans un cas où</b> une <b>offrande n'est pas assimilée à</b> une autre <b>offrande. Quel</b> est le sens de l'expression : Une <b>offrande n'est pas assimilée</b> à une autre <b>offrande ?</b> Quelles offrandes ne sont pas issues du même type d'animal, mais néanmoins les <i>halakhot</i> régissant l'aspersion de leur sang sont les mêmes ?

### Partie 9
<b>Si l'on dit</b> que Rabbi Yishmael fait référence au <b>taureau de Yom Kippour et au bouc de Yom Kippour,</b> et qu'il en déduit par voie de <i>a fortiori</i> que les actions concernant le taureau de Yom Kippour sont les mêmes que celles concernant le taureau pour un péché involontaire du prêtre oint, qui sont le même animal, cela <b>peut être réfuté</b> comme suit : <b>Qu'est-ce qui</b> est notable <b>dans ces</b> offrandes, le taureau et le bouc de Yom Kippour ? Elles sont remarquables <b>en ce que leur sang pénètre dans le sanctuaire le plus intérieur,</b> le Saint des Saints. Ce n'est pas le cas du taureau pour un péché involontaire du prêtre oint, dont le sang n'est aspergé que dans la zone extérieure du Sanctuaire.

### Partie 10
<b>Disons plutôt,</b> que la référence ici est au <b>taureau pour un péché communautaire involontaire et aux boucs pour</b> un péché communautaire involontaire d'<b>adoration d'idoles.</b> Mais une fois encore, cette affirmation <b>peut être réfutée : Qu'est-ce qui est notable <b>dans ces offrandes</b> ? Elles sont remarquables <b>en ce que</b> elles expient toutes deux <b>les transgressions d'une mitzva connue,</b> alors que le taureau de Yom Kippour expie des transgressions inconnues (voir <i>Shevuot</i> 2a).

### Partie 11
<b>La référence est plutôt,</b> le taureau pour un péché communautaire involontaire et le bouc de Yom Kippour, et c'est</b> ce que Rabbi Yishmael <b>dit : Tout comme dans un cas où</b> une <b>offrande n'est pas assimilée à</b> une autre <b>offrande, car celle-ci</b> est <b>un taureau et celle-là</b> est <b>un bouc, </b> néanmoins les <b>actions</b> d'aspersion du sang dans une offrande <b>sont assimilées</b> aux <b>actions</b> du sang dans l'autre offrande <b>en ce qui concerne ce qui est écrit à son sujet,</b> dans <b>un cas dans lequel</b> une <b>offrande est assimilée à</b> une autre <b>offrande,</b> c'est-à-dire. e., le taureau pour un péché involontaire du prêtre oint et le taureau de Yom Kippour, <b>comme ceci</b> est <b>un taureau et que</b> est <b>un taureau, n'est-ce pas logique</b>.
