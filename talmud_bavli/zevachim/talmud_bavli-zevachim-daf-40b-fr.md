En ce qui concerne le terme <b>"dans le sang"</b> (Levitique 4:6), il enseigne <b>que le sang</b> dans le recipient de service <b>doit etre d'une</b> <b>mesure suffisante pour l'immersion des le depart.</b> Le pretre doit initialement recueillir dans le recipient suffisamment de sang pour toutes les aspersions, plutot que d'ajouter du sang dans le recipient pour chaque aspersion. Enfin, l'expression <b>"et</b> le pretre <b>s'immergera</b> son doigt dans le sang indique qu'il doit y avoir suffisamment de sang dans le recipient pour que le pretre puisse y immerger son doigt <b>et ne pas</b> avoir a <b>essuyer</b> les cotes de l'ustensile pour recueillir le sang pour l'aspersion.

### Partie 2
La Gemara explique : <b>Et</b> il <b>etait necessaire</b> que le verset enonce ces deux derniers termes. Il fallait <b>ecrire " dans le sang ", comme si le Misericordieux avait ecrit</b> seulement <b>"et</b> le pretre <b>s'immergera, je dirais</b> que les aspersions sont valables <b>meme s'il n'y avait pas</b> une <b>mesure</b> suffisante de sang <b>pour une immersion des le depart,</b> mais seulement assez pour une seule aspersion, a condition que le pretre ajoute ensuite plus de sang au recipient pour chaque aspersion. Par consequent, <b>Le Misericordieux ecrit "dans le sang"</b> pour enseigner que, des le depart, il doit y avoir suffisamment de sang dans le recipient pour toutes les aspersions.

### Partie 3
<b>Et</b> inversement, <b>si le Misericordieux avait ecrit</b> seulement <b>"dans le sang", je dirais</b> que <b>meme</b> s'il y avait assez de sang dans le recipient au depart, il n'est pas necessaire qu'il y ait assez de sang pour l'immersion pour les dernieres aspersions, car le pretre peut <b>essuyer</b> les cotes de l'ustensile pour recueillir le sang pour l'aspersion. Par consequent, <b>Le Misericordieux ecrit "et</b> le pretre <b>doit immerger."</b>

### Partie 4
La Guemara aborde une autre phrase apparemment superflue dans le meme chapitre : "Et le prêtre mettra un peu de sang sur les coins de <b>l'autel des parfums odorants</b> devant le Seigneur" (Levitique 4:7). <b>Pourquoi ai-je</b> besoin du verset pour mentionner l'encens doux ? Il aurait suffi d'identifier l'autel comme etant "devant le Seigneur" et on aurait compris que la reference est a l'autel interieur. Cela sert plutèt A enseigner <b>que si l'autel n'avait pas été inauguré avec de l'encens doux,</b> le sacrificateur <b>n'y aurait pas aspergé du sang</b>.

### Partie 5
La Gemara commente : <b>Il est enseigne</b> dans une <i>baraita</i> <b>conformement</b> a l'avis <b>du Rav Pappa</b> que la phrase superflue dans le passage discutant du taureau pour un peche communautaire involontaire fait allusion au taureau de Yom Kippour afin d'enseigner les trois <i>halakhot</i> de <i>et</i>, dans le sang, et de l'immersion, a partir du cas du taureau pour un peche involontaire du pretre oint, auquel il est egalement fait allusion dans ce verset. Le <i>baraita</i> precise : <b>"Et il fera</b> avec le taureau, <b>comme il a fait</b> avec le taureau pour le sacrifice pour le peche" (Levitique 4:20). <b>Quel est le sens</b> lorsque le verset dit "avec le taureau" ? </b> Cela sert <b>a inclure le taureau de Yom Kippour pour tout ce qui est enonce dans</b> cette <b>matiere,</b> c'est-a-dire dans le passage concernant le taureau pour un peche involontaire du pretre oint, specifiquement les <i>halakhot</i> de <i>et</i>, dans le sang, et l'immersion. C'est <b>la declaration de Rabbi</b> Yehuda HaNasi.

### Partie 6
<b>Rabbi Yishmael a dit :</b> Cette inclusion est inutile, car ces <i>halakhot</i> peuvent etre derivees via <b>une <i>a fortiori</i></b> inference : <b>Comme dans un cas ou</b> une <b>offrande n'est pas assimilee a</b> une autre <b>offrande</b> d'un type d'animal different, la Torah a <b>egalise</b> les <b>actions</b> d'aspersion du sang dans une offrande <b>avec</b> les <b>actions</b> du sang dans l'autre offrande, comme cela sera explique, dans <b>un cas dans lequel</b> la Torah <b>egalise</b> une <b>offrande avec</b> une autre <b>offrande,</b> c'est-a-dire. e., le taureau pour un peche involontaire du pretre oint et le taureau de Yom Kippour, <b>n'est-il pas logique que</b> la Torah <b>assortisse</b> les <b>actions</b> d'aspersion du sang dans une offrande <b>avec</b> les <b>actions</b> du sang dans l'autre offrande ? Par consequent, la derivation par le biais d'une inclusion speciale n'est pas necessaire.

### Partie 7
<b>Au contraire, quel est le sens</b> lorsque <b>le verset dit :</b> "Et il fera avec le taureau, comme il a fait avec le taureau pour le sacrifice pour le peche" ? En ce qui concerne le premier cas de <b>"avec le taureau", il s'agit du taureau pour un peche communautaire involontaire.</b> Et en ce qui concerne le second cas de <b>"avec le taureau", il s'agit du taureau pour</b> un peche involontaire du <b>pretre oint.</b> Et le verset sert a enseigner que tout comme dans le premier cas, si le pretre a omis une des aspersions, il n'a rien fait, il en va de meme dans le second cas.

### Partie 8
La Guemara clarifie la <i>baraita</i>. <b>Le Maitre a dit</b> ci-dessus : <b>Comme dans un cas ou</b> une <b>offrande n'est pas assimilee a</b> une autre <b>offrande. Quel</b> est le sens de l'expression : Une <b>offrande n'est pas assimilee</b> a une autre <b>offrande ?</b> Quelles offrandes ne sont pas issues du meme type d'animal, mais neanmoins les <i>halakhot</i> regissant l'aspersion de leur sang sont les memes ?

### Partie 9
<b>Si l'on dit</b> que Rabbi Yishmael fait reference au <b>taureau de Yom Kippour et au bouc de Yom Kippour,</b> et qu'il en deduit par voie de <i>a fortiori</i> que les actions concernant le taureau de Yom Kippour sont les memes que celles concernant le taureau pour un peche involontaire du pretre oint, qui sont le meme animal, cela <b>peut etre refute</b> comme suit : <b>Qu'est-ce qui</b> est notable <b>dans ces</b> offrandes, le taureau et le bouc de Yom Kippour ? Elles sont remarquables <b>en ce que leur sang penetre dans le sanctuaire le plus interieur,</b> le Saint des Saints. Ce n'est pas le cas du taureau pour un peche involontaire du pretre oint, dont le sang n'est asperge que dans la zone exterieure du Sanctuaire.

### Partie 10
<b>Disons plutot,</b> que la reference ici est au <b>taureau pour un peche communautaire involontaire et aux boucs pour</b> un peche communautaire involontaire d'<b>adoration d'idoles.</b> Mais une fois encore, cette affirmation <b>peut etre refutee : Qu'est-ce qui est notable <b>dans ces offrandes</b> ? Elles sont remarquables <b>en ce que</b> elles expient toutes deux <b>les transgressions d'une mitzva connue,</b> alors que le taureau de Yom Kippour expie des transgressions inconnues (voir <i>Shevuot</i> 2a).

### Partie 11
<b>La reference est plutot,</b> le taureau pour un peche communautaire involontaire et le bouc de Yom Kippour, et c'est</b> ce que Rabbi Yishmael <b>dit : Tout comme dans un cas ou</b> une <b>offrande n'est pas assimilee a</b> une autre <b>offrande, car celle-ci</b> est <b>un taureau et celle-la</b> est <b>un bouc, </b> neanmoins les <b>actions</b> d'aspersion du sang dans une offrande <b>sont assimilees</b> aux <b>actions</b> du sang dans l'autre offrande <b>en ce qui concerne ce qui est ecrit a son sujet,</b> dans <b>un cas dans lequel</b> une <b>offrande est assimilee a</b> une autre <b>offrande,</b> c'est-a-dire. e., le taureau pour un peche involontaire du pretre oint et le taureau de Yom Kippour, <b>comme ceci</b> est <b>un taureau et que</b> est <b>un taureau, n'est-ce pas logique</b>.
